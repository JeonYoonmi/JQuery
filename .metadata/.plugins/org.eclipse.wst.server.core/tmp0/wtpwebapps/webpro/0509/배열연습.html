배열 : 여러개의 값을 일련의 연속적인 공간에 저장한다.
		배열의 자료형과 상관없이 모든 데이터를 저장할 수 있다.
		배열의 갯수는 자동으로 변경한다.
		생성 1 : fruit = ["사과", "바나나"]
	


		배열 : 여러개의 값을 일련의 연속적인 공간에 저장한다.
		생성 2 : fruit = new Array("사과", "바나나", "복숭아")
	


		배열 : 여러개의 값을 일련의 연속적인 공간에 저장한다.
		생성 3 : fruit = new Array(5)
		fruit[0] = "사과";
		fruit[1] = "바나나";
		fruit[2] = "바나나";
		fruit[3] = "바나나";
		fruit[4] = "바나나";
