스트립트에서 color값을 활용하여 색상을 적용시킨다.

스트립트로 선택한 값을 가져가기 위해서

태그 이름을 이용 :
document.getElementByTagName('input') document.querySelector('input') document.querySelectorAll('input')

name속성을 이용 :
document.ff.color

id속성을 이용 :
document.getElementById('color'); document.querySelector('#color');

class속성을 이용:
document.getElementsByClassName('color') document.querySelector('.color') document.querySelectorAll('.color')
