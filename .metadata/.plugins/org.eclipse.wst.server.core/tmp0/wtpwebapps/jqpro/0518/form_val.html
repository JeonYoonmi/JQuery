<button id="btnview">조사하기 </button>
		type이 생략된 button은 submit기능을 수행한다.
		
		전송이 필요없는 경우는
		 1. type="button"을 추가한다.
		 2. 함수 안에서 return false를 마지막 문장에 추가한다.
		 3. 함수 안에서 event.preventDefault()메소드호출 문장을 추가한다. (클릭해서 이벤트가 발생하므로 event라는 속성이 생긴다.)
		 event.preventDefault() : 요소들의 고유 기능의 수행을 방지 한다.
클릭하면 수를 입력하고 15이상일떄만 페이지 이동한다.
