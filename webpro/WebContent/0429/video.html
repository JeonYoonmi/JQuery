무음인 경우 항상 자동 재생이 가능.
	음소거 ON: <video> 요소에 muted 속성을 추가 하거나, video.muted 값을 true 로 설정한 경우는 자동 재생이 가능합니다.

   	구글에서 2018년에 Autoplay Policy를 재정의 했다.
   	SNS 및 여러 포스팅이 활발해 짐에 따라 용량이 큰 동영상 재생등이 사용자가 원치않는 리소스를 가져간다고 판단하여 브라우저에서 막아버렸다.

  	#1 <video> 요소에 muted 속성 추가
   	  <video autoplay muted></video>
  
 	#2 video.muted 값을 true로 설정
   	  var video = document.querySelector('video');
      video.muted = true; // 
   
	동일 origin 도메인 내에서 유저 인터렉션 이벤트 발생 시 유음 자동 재생. (Chrome)- youtube
